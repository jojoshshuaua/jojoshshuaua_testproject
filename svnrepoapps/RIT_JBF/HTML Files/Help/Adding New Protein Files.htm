Adding New Protein Files

1) Enter ftp://ftp.ncbi.nih.gov/genomes/Bacteria/ into your web browser’s url bar to navigate to this page:

2) Double click on the folder of any bacterium.

3) Click on a file in the bacterium’s folder that has the extension .gbk or .faa.

4) The browser will now display a text page showing the file’s contents. Double check that the file contains protein sequences, not DNA or RNA sequences.

5) Save the web page (via the File menu) to the Data folder in the JBioFramework folder.

6) Restart JBioFramework if it is currently running.

7) The new .faa or .gbk file will now be accessible inside the simulations of JBioFramework.
