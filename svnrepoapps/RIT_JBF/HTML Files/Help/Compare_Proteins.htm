Compare Proteins

Click on the Compare Proteins button when a gel has already run to bring up another load protein data file window.

This will allow you to add another set of proteins from any of the data files to compare to the proteins already added. Common proteins between organisms can be identified, as well as which proteins are unique to one organism or the other.

Proteins in common are colored green, proteins belonging only to the first data file loaded are colored red, and proteins belonging only to the second data file loaded are colored yellow.
