Isoelectric Focusing

Once your proteins have been loaded, IEF should be selected from the "Animation Chooser" drop down menu. It is not possible to skip directly to SDS-PAGE animiation without first running the IEF animation.

Pushing the run button on the "Animation Buttons" will start the isoelectric focusing simulation similar to below. After running this changing the separation conditions will have no effect.
