Display Protein List

Clicking on the Display Protein List button will bring up the Protein Lists Frame which contains all of the proteins (ordered by accession number)in the file or files (via the Compare Proteins button) that are currently loaded:

You can manipulate which proteins are displayed on the gel canvas by editing the proteins displayed in the two sequence lists. The buttons on the bottom of the Protein Lists Frame will remove all the proteins from sequence one, sequence two, the common proteins, or the proteins you have selected.

To view your changes to the protein lists on the gel canvas, click the Stop button followed by the Run button.
